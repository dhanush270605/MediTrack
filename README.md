# MediTrack

MediTrack – Smart Medication Management System

## 📌 Project Description

MediTrack is a web-based application designed to help users manage medications, schedules, and adherence tracking.

## 🚀 Features
- User Login & Signup
- Medication Management
- Scheduling & Reminders
- Adherence Tracking
- Caregiver Support

## 🛠 Tech Stack
- Frontend: Next.js / React
- Backend: Node.js (Express)
- Database: PostgreSQL
- Cloud: AWS (EC2, RDS, S3)

## 📂 Project Structure
- `frontend/` → UI application
- `backend/` → API server
- `database/` → SQL scripts
- `docs/` → diagrams

## ▶️ How to Run
**Backend**
```bash
cd backend
npm install
node server.js
```

**Frontend**
```bash
cd frontend
npm install
npm run dev
```
